# Cat Café Diary ☕📖

Static multi-page website (HTML/CSS/JS) with a kawaii café vibe and AR/EN toggle.

## Run

- Open `index.html` in any browser.

Pages:
- `index.html` (Home)
- `diary.html` (Diary + memories)
- `cats.html` (My Cats)
- `contact.html` (Contact)

## Customize (quick)

- Change the name in `index.html` (search for `profileName`).
- Replace the `mailto:` email in the Contact section.
- Replace SVGs inside `assets/` if you want different cats/stickers.

## Deploy

You can upload the folder to any static host (Netlify, GitHub Pages, etc.).

> Note: This is a static site (no server). All content is editable in the files.
